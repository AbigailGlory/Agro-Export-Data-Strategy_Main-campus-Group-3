# 🌿 Agro-Export Data Strategy Dashboard

A data strategy presentation dashboard built with Streamlit, developed by 
Main Campus Group 3 as part of an academic supervised project.

---

## 📋 Project Overview

This interactive dashboard presents a comprehensive 24-month data strategy 
for an agro-export company supplying onions, mangoes, and cashew products 
to international markets. The strategy addresses key challenges including 
yield variability, post-harvest losses, and export rejections by treating 
data as a core business asset.

## 📑 Dashboard Sections

1. **Overview** — Strategy summary and key metrics at a glance
2. **Executive Summary** — Core challenges and current baseline numbers
3. **Organizational Context** — Industry landscape and business goals
4. **Business Questions** — Critical decisions the strategy must answer
5. **Analytics Maturity** — Current and target analytics capability levels
6. **Target Data Vision** — Measurable goals across all crop types
7. **Data Architecture** — Technology stack and data flow diagram
8. **Governance & Privacy** — Data ownership and regulatory compliance
9. **Risks & Ethics** — Ethical considerations and risk mitigation
10. **KPIs & Impact** — Success metrics and business impact projections
11. **Productivity, Quality & Traceability** — How the strategy drives results
12. **Thank You** — Final summary and 24-month impact overview

## 🚀 Deployment

This app is deployed on Streamlit Community Cloud and is accessible via 
a public URL. It connects directly to this GitHub repository and 
automatically redeploys whenever changes are pushed to the main branch.

### Run Locally

To run the app on your local machine:

1. Clone the repository
```
git clone https://github.com/YOUR_USERNAME/agro-export-strategy.git
cd agro-export-strategy
```

2. Install dependencies
```
pip install -r requirements.txt
```

3. Run the app
```
streamlit run AgroExportStrategy.py
```

4. Open your browser at http://localhost:8501

## 📊 Key Strategy Targets (24 Months)
